```rust
/// Configuration for context packing
#[derive(Debug, Clone)]
pub struct PackerConfig {
    /// Maximum tokens in output (default 4000)
    pub max_tokens: usize,
    /// Approximate chars per token (default 4)
    pub chars_per_token: usize,
    /// MMR lambda: 0 = max diversity, 1 = max relevance (default 0.7)
    pub mmr_lambda: f32,
    /// Similarity threshold for deduplication (default 0.9)
    pub dedup_threshold: f32,
    /// Minimum chunks per type for diversity (default 1)
    pub min_per_type: usize,
}

/// Input chunk for packing
#[derive(Debug, Clone)]
pub struct PackerInput {
    pub chunk_id: ChunkId,
    pub text: String,
    pub chunk_type: ChunkType,
    pub score: f32,
    pub hash: String,
    pub embedding: Option<Vec<f32>>,  // For MMR similarity
    pub source_uri: Option<String>,
}

/// Output packed chunk with metadata
#[derive(Debug, Clone)]
pub struct PackedChunk {
    pub chunk_id: ChunkId,
    pub text: String,
    pub chunk_type: ChunkType,
    pub score: f32,
    pub source_uri: Option<String>,
    pub token_count: usize,
}

/// Result of packing
#[derive(Debug)]
pub struct PackedContext {
    pub chunks: Vec<PackedChunk>,
    pub total_tokens: usize,
    pub duplicates_removed: usize,
    pub diversity_adjustments: usize,
}
```

Implement ContextPacker:

```rust
pub struct ContextPacker {
    config: PackerConfig,
}

impl ContextPacker {
    pub fn new(config: PackerConfig) -> Self;

    /// Pack chunks with deduplication, MMR, and token budgeting
    pub fn pack(&self, mut chunks: Vec<PackerInput>) -> PackedContext;
}
```

**Algorithm:**

1. **Deduplication (hash-based):**
   - Sort by score descending
   - Track seen hashes in HashSet
   - Skip chunks with duplicate hash
   - Count duplicates_removed

2. **Deduplication (similarity-based, if embeddings available):**
   - For each candidate, check cosine similarity to already-selected
   - Skip if any selected chunk has similarity > dedup_threshold
   - Only applies if embedding is Some

3. **MMR Selection:**
   - Start with highest-scoring chunk
   - For each remaining slot:
     - Score = lambda * relevance + (1-lambda) * diversity
     - relevance = original score (normalized)
     - diversity = min distance to all selected chunks (if embeddings)
     - If no embeddings, use type diversity: bonus if different type
   - Select chunk with highest MMR score
   - Track diversity_adjustments when diversity changes selection

4. **Type diversity enforcement:**
   - Track count per ChunkType
   - After initial selection, if any type has 0 chunks:
     - Find highest-scoring chunk of that type
     - Swap with lowest-scoring of over-represented type
   - Only apply if min_per_type > 0

5. **Token budgeting:**
   - Estimate tokens = text.len() / chars_per_token
   - Accumulate tokens as chunks added
   - Stop when next chunk would exceed max_tokens
   - Truncate final chunk if close to limit (optional: skip instead)

6. **Build output:**
   - Create PackedChunk for each selected
   - Return PackedContext with totals

**Cosine similarity helper:**
```rust
fn cosine_similarity(a: &[f32], b: &[f32]) -> f32 {
    let dot: f32 = a.iter().zip(b).map(|(x, y)| x * y).sum();
    let norm_a: f32 = a.iter().map(|x| x * x).sum::<f32>().sqrt();
    let norm_b: f32 = b.iter().map(|x| x * x).sum::<f32>().sqrt();
    if norm_a == 0.0 || norm_b == 0.0 { 0.0 } else { dot / (norm_a * norm_b) }
}
```
