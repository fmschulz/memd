#[derive(Debug, Clone)]
pub struct HybridConfig {
    /// Number of candidates to fetch from dense search
    pub dense_k: usize,  // default 100
    /// Number of candidates to fetch from sparse search
    pub sparse_k: usize,  // default 100

    /// Packer configuration

}

**HybridSearcher:**
```rust
pub struct HybridSearcher {
    dense: Arc<DenseSearcher>,
    sparse: Option<Arc<Bm25Index>>,
    text_processor: TextProcessor,
    fusion: RrfFusion,
    reranker: FeatureReranker,
    packer: ContextPacker,
    config: HybridConfig,
}

impl HybridSearcher {
    pub fn new(
        dense: Arc<DenseSearcher>,
        sparse: Option<Arc<Bm25Index>>,
        config: HybridConfig,
    ) -> Self;

    /// Index a chunk in both dense and sparse indexes
    pub async fn index_chunk(
        &self,
        tenant_id: &TenantId,
        chunk_id: &ChunkId,
        text: &str,
    ) -> Result<()>;

    /// Remove chunk from indexes
    pub fn delete_chunk(
        &self,
        tenant_id: &TenantId,
        chunk_id: &ChunkId,
    ) -> Result<()>;

    /// Perform hybrid search with fusion and reranking
    pub async fn search(
        &self,
        tenant_id: &TenantId,
        query: &str,
        k: usize,
        context: Option<SearchContext>,
    ) -> Result<Vec<HybridSearchResult>>;

    /// Search with timing information for metrics
    pub async fn search_with_timing(
        &self,
        tenant_id: &TenantId,
        query: &str,
        k: usize,
        context: Option<SearchContext>,
    ) -> Result<(Vec<HybridSearchResult>, HybridTiming)>;
}

/// Context for search (project, preferences)
pub struct SearchContext {
    pub current_project: Option<String>,
    pub preferred_types: Vec<ChunkType>,
}

/// Timing breakdown
pub struct HybridTiming {
    pub dense_time: Duration,
    pub sparse_time: Duration,
    pub fusion_time: Duration,
    pub rerank_time: Duration,
    pub total_time: Duration,
}
```

**Search algorithm:**

1. **Parallel candidate generation:**
   - Dense: search(tenant_id, query, dense_k)
   - Sparse: search(tenant_id, query, sparse_k) if enabled

2. **Build fusion candidates:**
   - Convert dense results to FusionCandidate with source=Dense
   - Convert sparse results to FusionCandidate with source=Sparse

3. **Fuse with RRF:**
   - fusion.fuse(all_candidates) -> Vec<FusedResult>

4. **Rerank (if context available):**
   - Load chunk metadata for each fused result
   - Build ChunkWithMeta for each
   - reranker.rerank(chunks, context) -> Vec<RankedResult>

5. **Return top K:**
   - Take first k results
   - Convert to HybridSearchResult
