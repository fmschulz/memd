**Struct SymbolQueryService:**
- store: Arc(StructuralStore)

**Query Result Types:**

Struct SymbolLocation:
- file_path: String
- name: String
- kind: SymbolKind
- line_start: u32
- line_end: u32
- col_start: u32
- col_end: u32
- signature: Option(String)
- docstring: Option(String)
- visibility: Option(String)
- language: String

Struct CallerInfo:
- caller_name: String
- caller_file: String
- call_line: u32
- call_col: u32
- caller_kind: SymbolKind

Struct ImportInfo:
- importing_file: String
- import_line: u32
- alias: Option(String)

**Impl SymbolQueryService:**

- new(store: Arc(StructuralStore)) -> Self

- find_symbol_definition(&self, tenant_id: &TenantId, name: &str, project_id: Option(&str)) -> Result(Vec(SymbolLocation))
  1. Query symbols table by name
  2. Filter by project_id if provided
  3. Order by kind priority (function > method > variable > type)
  4. Return locations with full metadata

- find_references(&self, tenant_id: &TenantId, name: &str, project_id: Option(&str)) -> Result(Vec(SymbolLocation))
  1. Find symbol definitions by name (the definition itself is a reference)
  2. Find call edges where callee_name matches
  3. Combine and deduplicate by location
  4. Return all locations where the name appears

- find_callers(&self, tenant_id: &TenantId, name: &str, max_depth: u32, project_id: Option(&str)) -> Result(Vec(CallerInfo))
  1. Find call edges where callee_name matches
  2. For each edge, get caller symbol info
  3. If max_depth > 1, recursively find callers of callers (with cycle detection)
  4. Use recursive CTE in SQL for efficiency:
     ```sql
     WITH RECURSIVE callers AS (
       SELECT caller_symbol_id, 1 as depth FROM call_edges WHERE callee_name = ?
       UNION ALL
       SELECT ce.caller_symbol_id, c.depth + 1 FROM call_edges ce
       JOIN callers c ON ce.callee_symbol_id = c.caller_symbol_id
       WHERE c.depth < ?
     )
     ```
  5. Return caller info with depth

- find_imports(&self, tenant_id: &TenantId, module: &str, project_id: Option(&str)) -> Result(Vec(ImportInfo))
  1. Query imports table by imported_module
  2. Return files that import this module with line info

**Helper methods:**

- link_callees(&self, tenant_id: &TenantId) -> Result(usize)
  - Attempt to resolve unresolved callee_symbol_id by matching callee_name to symbol names
  - Called after indexing new files
  - Returns count of newly linked edges
