**Enum QueryIntent:**
```rust
pub enum QueryIntent {
    // Semantic search (default)
    SemanticSearch,

    // Structural code queries
    SymbolDefinition(String),   // Find where X is defined
    SymbolReferences(String),   // Find usages of X
    SymbolCallers(String),      // Find who calls X
    ModuleImports(String),      // Find who imports X
    FileSymbols(String),        // All symbols in file X

    // Debug/trace queries
    ToolCalls(Option<String>),  // Tool calls, optionally filtered
    ErrorSearch(Option<String>), // Errors, optionally by signature

    // Document queries
    DocQa,                      // Documentation question
    DecisionWhy(String),        // Why was X decided
    PlanNext,                   // What's the next step
}
```

**Struct QueryRouter:**
- definition_patterns: Vec(CompiledPattern)
- caller_patterns: Vec(CompiledPattern)
- reference_patterns: Vec(CompiledPattern)
- import_patterns: Vec(CompiledPattern)
- error_patterns: Vec(CompiledPattern)
- tool_patterns: Vec(CompiledPattern)

**Struct CompiledPattern:**
- regex: Regex
- extract_group: usize  -- Which capture group contains the target

**Impl QueryRouter:**

- new() -> Self
  Initialize all patterns:

  Definition patterns:
  - `(?i)where is (\w+) defined`
  - `(?i)definition of (\w+)`
  - `(?i)find (?:class|function|type) (\w+)`
  - `(?i)what is (\w+)`
  - `(?i)show (?:me )?(\w+) (?:definition|source)`

  Caller patterns:
  - `(?i)who calls (\w+)`
  - `(?i)callers of (\w+)`
  - `(?i)what calls (\w+)`
  - `(?i)(\w+) is called by`
  - `(?i)find callers (?:of )?(\w+)`

  Reference patterns:
  - `(?i)references to (\w+)`
  - `(?i)usages of (\w+)`
  - `(?i)where is (\w+) used`
  - `(?i)find all (?:uses|usages|references) (?:of )?(\w+)`

  Import patterns:
  - `(?i)who imports (\w+)`
  - `(?i)files (?:that )?import (\w+)`
  - `(?i)(\w+) importers`

  Error patterns:
  - `(?i)errors? (?:in|from|about) (\w+)`
  - `(?i)stack traces? (?:for|with) (\w+)`
  - `(?i)exceptions? (?:in|from) (\w+)`
  - `(?i)what went wrong`  -- general error search

  Tool patterns:
  - `(?i)(?:tool )?calls to (\w+)`
  - `(?i)when did (?:we|I) (?:call|use) (\w+)`
  - `(?i)history of (\w+) calls`
  - `(?i)recent tool calls`

- classify(&self, query: &str) -> QueryIntent
  1. Check explicit prefixes first (override pattern detection):
     - "def:" -> SymbolDefinition
     - "callers:" -> SymbolCallers
     - "refs:" -> SymbolReferences
     - "imports:" -> ModuleImports
     - "errors:" -> ErrorSearch
     - "tools:" -> ToolCalls

  2. Try each pattern category in priority order:
     - Definition patterns
     - Caller patterns
     - Reference patterns
     - Import patterns
     - Error patterns
     - Tool patterns

  3. If no pattern matches, return SemanticSearch

- extract_target(pattern: &Regex, query: &str, group: usize) -> Option(String)
  - Run regex, extract capture group
  - Normalize target (trim, etc.)

**Struct RouteResult:**

- confidence: f32  -- How confident in classification (pattern match = 0.9, prefix = 1.0)

1. Classify query with router
2. Based on intent:

   SymbolDefinition(name):
   - Call symbol_query_service.find_symbol_definition(name)
   - If results found, return StructuralResult
   - If empty, fall back to semantic search

   SymbolCallers(name):
   - Call symbol_query_service.find_callers(name, depth=1)
   - Return StructuralResult or fallback

   SymbolReferences(name):
   - Call symbol_query_service.find_references(name)
   - Return StructuralResult or fallback

   ModuleImports(module):
   - Call symbol_query_service.find_imports(module)
   - Return StructuralResult or fallback

   ToolCalls(name):
   - Call query_service.find_tool_calls()
   - Return TraceResult

   ErrorSearch(sig):
   - Call query_service.find_errors()
   - Return TraceResult

   SemanticSearch (default):
   - Use existing hybrid_search()
   - Return HybridResult
