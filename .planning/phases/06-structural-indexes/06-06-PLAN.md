Extend crates/memd/src/structural/queries.rs with trace queries:

**Query Result Types:**

Struct ToolCallResult:
- trace_id: i64
- tool_name: String
- timestamp_ms: i64
- timestamp_formatted: String  -- ISO 8601 formatted
- input: Option(serde_json::Value)
- output: Option(serde_json::Value)
- error: Option(serde_json::Value)
- duration_ms: Option(i64)
- session_id: Option(String)
- context_tags: Vec(String)

Struct ErrorResult:
- trace_id: i64
- error_signature: String
- error_message: String
- timestamp_ms: i64
- timestamp_formatted: String
- frames: Vec(FrameInfo)
- session_id: Option(String)

Struct FrameInfo:
- index: u32
- function_name: Option(String)
- file_path: Option(String)
- line_number: Option(u32)
- col_number: Option(u32)

**Add to SymbolQueryService impl:**

- find_tool_calls(&self, tenant_id: &TenantId, tool_name: Option(&str), time_range: Option(TimeRange), session_id: Option(&str), limit: usize) -> Result(Vec(ToolCallResult))
  1. Query tool_traces with optional filters
  2. Parse JSON fields back to Value
  3. Format timestamps for display
  4. Order by timestamp DESC
  5. Apply limit

- find_tool_calls_with_errors(&self, tenant_id: &TenantId, time_range: Option(TimeRange)) -> Result(Vec(ToolCallResult))
  - Find tool calls where error_json IS NOT NULL
  - Useful for debugging

- find_errors(&self, tenant_id: &TenantId, error_signature: Option(&str), function_name: Option(&str), file_path: Option(&str), time_range: Option(TimeRange), limit: usize) -> Result(Vec(ErrorResult))
  1. Query stack_traces with filters
  2. For each trace, fetch frames
  3. Return complete error context
  4. Order by timestamp DESC

- find_errors_in_function(&self, tenant_id: &TenantId, function_name: &str) -> Result(Vec(ErrorResult))
  - Find errors where function appears in stack
  - Uses stack_frames.function_name index

- get_error_summary(&self, tenant_id: &TenantId, time_range: Option(TimeRange)) -> Result(Vec(ErrorSummary))
  - Group errors by error_signature
  - Return count, first_seen, last_seen for each

Struct ErrorSummary:
- error_signature: String
- count: u64
- first_seen_ms: i64
- last_seen_ms: i64

**Helper functions:**
- format_timestamp(ms: i64) -> String
  - Convert Unix ms to ISO 8601 string

Add unit tests:
- test_find_tool_calls_by_name
- test_find_tool_calls_by_time_range
