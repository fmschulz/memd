Create evals/harness/src/suites/structural.rs:

**Struct StructuralSuite:**
- mcp_client: McpClient
- dataset: StructuralDataset

**Struct StructuralDataset:**
- test_project: TestProject
- queries: Vec(StructuralQuery)
- natural_language_queries: Vec(NLQuery)

**Impl StructuralSuite:**

- load(dataset_path: &Path) -> Result(Self)
  - Parse JSON dataset
  - Initialize MCP client

- setup(&self, tenant_id: &TenantId) -> Result(())
  1. Create test project in memd
  2. Index all test files via memory.add
  3. Wait for indexing to complete
  4. Run structural indexer on files

- run_definition_tests(&self, tenant_id: &TenantId) -> Vec(TestResult)
  For each find_definition query:
  1. Call code.find_definition via MCP
  2. Compare result to expected (file, line, kind)
  3. Score: exact match = 1.0, file match = 0.5, no match = 0.0

- run_callers_tests(&self, tenant_id: &TenantId) -> Vec(TestResult)
  For each find_callers query:
  1. Call code.find_callers via MCP
  2. Compare callers to expected_callers
  3. Score: precision and recall of caller set

- run_references_tests(&self, tenant_id: &TenantId) -> Vec(TestResult)
  For each find_references query:
  1. Call code.find_references via MCP
  2. Check count matches expected_count
  3. Check files match expected_files

- run_imports_tests(&self, tenant_id: &TenantId) -> Vec(TestResult)
  For each find_imports query:
  1. Call code.find_imports via MCP
  2. Compare importers to expected_importers

- run_intent_classification_tests(&self) -> Vec(TestResult)
  For each natural_language_query:
  1. Create QueryRouter
  2. Classify query
  3. Compare intent to expected_intent
  4. Compare target to expected_target if applicable

- run_all(&self, tenant_id: &TenantId) -> StructuralSuiteResults
  - Run all test categories
  - Aggregate results

**Struct TestResult:**
- query_id: String
- passed: bool
- score: f32  -- 0.0 to 1.0
- expected: String
- actual: String
- duration_ms: u64

**Struct StructuralSuiteResults:**
- definition_tests: Vec(TestResult)
- callers_tests: Vec(TestResult)
- references_tests: Vec(TestResult)
- imports_tests: Vec(TestResult)
- intent_tests: Vec(TestResult)
- overall_score: f32
- pass_rate: f32

**Metrics:**
- Precision: correct / returned
- Recall: correct / expected
- F1: harmonic mean of precision and recall

```rust
pub async fn run_suite_b(config: &EvalConfig) -> SuiteBResults {
    let mut results = SuiteBResults::default();

    // Existing retrieval quality tests
    results.retrieval = run_retrieval_suite(config).await?;
    results.hybrid = run_hybrid_suite(config).await?;

    // Add structural tests (new)
    if config.include_structural {
        let structural_suite = StructuralSuite::load(&config.structural_dataset)?;
        structural_suite.setup(&config.tenant_id).await?;
        results.structural = structural_suite.run_all(&config.tenant_id).await?;
    }

    // Aggregate scores
    results.overall_score = calculate_overall_score(&results);

    results
}
```
