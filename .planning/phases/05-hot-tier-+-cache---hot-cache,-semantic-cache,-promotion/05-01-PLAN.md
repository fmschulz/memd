**tiered/access_tracker.rs - AccessTracker:**

Struct AccessEvent:
- chunk_id: ChunkId
- timestamp: i64 (Unix ms)
- project_id: Option(String)
- query_id: Option(String) -- for dedup repeated queries

Struct AccessStats:
- access_count: u32
- last_access: i64 (Unix ms)
- first_access: i64
- project_hits: HashMap(String, u32) -- count per project
- decay_score: f32 -- calculated on read

Struct PromotionScore:
- chunk_id: ChunkId
- score: f32 (0.0-1.0 normalized)
- frequency_component: f32
- recency_component: f32
- project_component: f32
- eligible: bool -- meets minimum threshold

Struct AccessTrackerConfig:
- frequency_weight: f32 (default 0.4)
- recency_weight: f32 (default 0.4)
- project_weight: f32 (default 0.2)
- decay_half_life_hours: u32 (default 24)
- min_accesses_for_promotion: u32 (default 2)

Impl AccessTracker:
- new(config) -> Self with HashMap(ChunkId -> AccessStats)
- record_access(event: AccessEvent) -- update stats
- get_promotion_score(chunk_id, current_project: Option) -> PromotionScore
- get_top_candidates(k: usize, current_project: Option) -> Vec(PromotionScore)
- decay_all() -- apply time decay to all entries (called periodically)
- remove_chunk(chunk_id) -- cleanup on deletion

Scoring formula:
- frequency = log2(access_count + 1) / max_log2_count (normalized 0-1)
- recency = exp(-hours_since_last_access / decay_half_life)
- project = 1.0 if chunk was accessed from current_project, 0.0 otherwise
- total = frequency * w_f + recency * w_r + project * w_p

Use parking_lot::RwLock for thread safety.

Update crates/memd/src/lib.rs to export tiered module.

Struct HotTierConfig:
- capacity_percentage: f32 (default 0.10 -- 10% of total chunks)
- max_chunks: usize (default 50_000 -- hard cap)
- min_promotion_score: f32 (default 0.3)
- eviction_check_interval: Duration (default 60s)
- hnsw_config: HnswConfig (smaller ef_search for speed)

Struct HotTier:
- index: HnswIndex (separate from warm tier)
- chunks: HashMap(ChunkId -> HotChunkEntry)
- embeddings: HashMap(ChunkId -> Vec(f32)) -- store for re-insertion
- config: HotTierConfig
- version: AtomicU64 -- for cache invalidation
- access_tracker: Arc(RwLock(AccessTracker))

Struct HotChunkEntry:
- chunk_id: ChunkId
- embedding: Vec(f32)
- promoted_at: i64
- promotion_score: f32
- tenant_id: TenantId

Impl HotTier:
- new(config, access_tracker) -> Self
- with_embedder(embedder: Arc(dyn Embedder)) -- for re-embedding if needed
- promote(chunk_id, embedding, tenant_id) -> Result(bool) -- true if added
- demote(chunk_id) -> bool -- remove from hot tier
- contains(chunk_id) -> bool
- search(query_embedding, k) -> Vec(SearchResult)
- search_with_timing(query_embedding, k) -> (Vec(SearchResult), Duration)
- len() -> usize
- version() -> u64
- evict_if_needed(total_indexed: usize) -- evict low-score chunks if over capacity
- get_stats() -> HotTierStats

Struct HotTierStats:
- chunk_count: usize
- capacity_used: f32 (percentage)
- version: u64
- avg_promotion_score: f32

HNSW config for hot tier:
- Use smaller ef_search (30 vs 50) for faster queries
- Same dimension as warm tier
- max_elements based on capacity_percentage

Record access on every search hit for automatic tracking.
