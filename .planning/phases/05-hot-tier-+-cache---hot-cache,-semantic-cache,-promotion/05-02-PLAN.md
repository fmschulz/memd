Create crates/memd/src/tiered/semantic_cache.rs:

Struct SemanticCacheConfig:
- similarity_threshold: f32 (default 0.85 -- cosine similarity for cache hit)
- ttl_seconds: u64 (default 2700 -- 45 min, middle of 30-60 range)
- max_entries: u64 (default 10_000)
- confidence_boost_on_hit: f32 (default 0.1)
- confidence_decay_rate: f32 (default 0.05 per hour)

Struct CacheEntry:
- query_embedding: Vec(f32)
- results: Vec(CachedResult)
- tenant_id: TenantId
- project_id: Option(String)
- memory_version: u64 -- version watermark at cache time
- created_at: i64 (Unix ms)
- last_hit: i64 (Unix ms)
- hit_count: u32
- confidence: f32 (0.0-1.0)

Struct CachedResult:
- chunk_id: ChunkId
- score: f32
- text_preview: String (first 200 chars for quick display)

Struct CacheHit:
- results: Vec(CachedResult)
- similarity: f32
- confidence: f32
- age_ms: u64
- cache_key: String

Struct SemanticCache:
- entries: moka::sync::Cache(String -> CacheEntry) -- keyed by hash of tenant + project
- query_index: RwLock(Vec(QueryIndexEntry)) -- for similarity search
- config: SemanticCacheConfig
- stats: CacheStats (atomic counters)

Struct QueryIndexEntry:
- cache_key: String
- embedding: Vec(f32)
- tenant_id: TenantId
- project_id: Option(String)

Impl SemanticCache:
- new(config) -> Self
- lookup(query_embedding, tenant_id, project_id, current_version) -> Option(CacheHit)
  - Iterate query_index filtered by tenant/project
  - Compute cosine similarity for each
  - Return first entry above threshold AND version >= current_version
  - Increment hit_count and confidence on hit
  - Return None if no match or version stale
- insert(query_embedding, tenant_id, project_id, results, memory_version)
  - Generate cache_key from tenant + project + embedding hash
  - Create CacheEntry with initial confidence 0.5
  - Add to moka cache (TTL handled by moka)
  - Add to query_index
- invalidate_tenant(tenant_id) -- remove all entries for tenant
- invalidate_by_version(tenant_id, min_version) -- remove stale entries
- invalidate_chunks(chunk_ids: Vec) -- remove entries containing these chunks
- get_stats() -> CacheStats
- prune_index() -- remove entries no longer in moka cache

Struct CacheStats:
- total_lookups: u64
- cache_hits: u64
- cache_misses: u64
- version_invalidations: u64
- ttl_expirations: u64
- avg_confidence: f32
- entry_count: usize

Helper cosine_similarity(a, b) -> f32:
- Dot product / (norm_a * norm_b)
- Handle zero vectors gracefully

Helper embedding_hash(embedding) -> String:
- Use first 8 bytes of SHA-256 of embedding bytes
- For cache key generation
