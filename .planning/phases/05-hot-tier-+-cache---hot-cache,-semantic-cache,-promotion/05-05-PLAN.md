Create evals/harness/src/suites/tiered.rs:

use existing patterns from retrieval.rs and hybrid.rs.

Struct TieredEvalConfig:
- dataset_path: PathBuf
- hot_tier_latency_threshold_ms: u64 (default 20)
- warm_tier_latency_threshold_ms: u64 (default 100)
- min_cache_hit_rate: f32 (default 0.8)
- warmup_queries: usize (default 50)

Struct TieredEvalResult:
- cache_tests: Vec(CacheTestResult)
- hot_tier_tests: Vec(TierTestResult)
- warm_tier_tests: Vec(TierTestResult)
- overall_pass: bool
- summary: String

Struct CacheTestResult:
- query_id: String
- repeat_count: usize
- cache_hits: usize
- avg_latency_ms: f64
- pass: bool

Struct TierTestResult:
- query_id: String
- expected_tier: String
- actual_tier: String
- latency_ms: u64
- recall_at_k: f32
- pass: bool

Impl TieredEvalSuite:
- new(config) -> Self
- async run(&self, client: &McpClient) -> Result(TieredEvalResult)

run() implementation:
1. Load dataset from tiered_eval.json
2. Ingest all documents via memory.add
3. Warmup phase: run warmup_queries to populate access patterns
   - Query hot tier documents multiple times to trigger promotion
4. Cache hit tests:
   - For each cache test query:
     - Run query repeat_count times
     - Verify later queries hit cache (using debug_tiers output)
     - Measure latency reduction
5. Hot tier latency tests:
   - For queries marked expected_tier="hot":
     - Run query with debug_tiers=true
     - Verify source_tier is "hot" or "cache"
     - Verify latency < hot_tier_latency_threshold_ms
6. Warm tier baseline tests:
   - For queries marked expected_tier="warm":
     - Run query with debug_tiers=true
     - Verify results found (recall check)
     - Record latency as baseline
7. Aggregate results:
   - Cache hit rate across all cache tests
   - Hot tier success rate
   - Latency comparison (hot vs warm)

Update evals/harness/src/suites/mod.rs:
- Add pub mod tiered;
- Add TieredEvalSuite to exports

Quality thresholds:
- Cache hit rate >= 0.8 for repeated queries
- Hot tier p50 latency < warm tier p50 latency (significant)
- All hot tier expected queries return from hot tier
