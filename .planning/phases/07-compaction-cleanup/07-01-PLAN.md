   - `paths_audited: Vec<String>` - list of code paths checked
   - `passed: bool` - true if no leaks found
   - Derive Debug, Clone

2. `TombstoneAudit` struct (stateless, just configuration):
   - No fields needed for now

3. Implement `TombstoneAudit`:
   - `fn new() -> Self`

   - `fn audit_segment_reader(&self, reader: &SegmentReader, metadata: &SqliteMetadataStore, segment_id: u64) -> Result<AuditResult>`
     Logic:
     - Get all metadata for segment via `metadata.get_by_segment(segment_id)`
     - For each deleted chunk (status == Deleted):
       - Try to read via `reader.read_chunk(ordinal)`
       - If it returns Some (data), add to tombstone_leaks (filtering failed!)
       - If it returns None, filtering is working correctly
     - Return AuditResult with paths_audited = ["SegmentReader::read_chunk"]

   - `fn audit_metadata_store(&self, metadata: &SqliteMetadataStore, tenant_id: &TenantId) -> Result<AuditResult>`
     Logic:
     - Call `metadata.list(tenant_id, 10000, 0)`
     - Verify no results have status == Deleted
     - Return AuditResult (should always pass since list() filters deleted)

4. Add `AuditResult::combine(results: Vec<AuditResult>) -> AuditResult` to merge multiple audit results
