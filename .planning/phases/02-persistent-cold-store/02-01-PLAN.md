Create the segment module with format definitions and append-only writer.

**crates/memd/src/store/segment/mod.rs:**
```rust
//! Segment storage module
//!
//! Segments are append-only files containing chunk payloads.
//! Directory structure: tenants/<tenant_id>/segments/seg_<id>/

pub mod format;
pub mod writer;

pub use format::{PayloadIndexRecord, SegmentMeta, SEGMENT_MAGIC};
pub use writer::SegmentWriter;
```

**crates/memd/src/store/segment/format.rs:**
- Define SEGMENT_MAGIC = *b"MSEG" (4 bytes)
- PayloadIndexRecord: offset (u64), length (u32), crc32 (u32) - 16 bytes total, #[repr(C)]
- SegmentMeta: id (u64), chunk_count (u32), created_ts (i64), finalized (bool)
- Implement serialization using byteorder for fixed-size records
- Add unit tests for roundtrip serialization

**crates/memd/src/store/segment/writer.rs:**
- SegmentWriter struct with: id, dir path, payload file, payload_len, index_records vec
- SegmentWriter::create(base_dir, segment_id) -> Result<Self>
  - Create directory: seg_{id:06} (zero-padded)
  - Open payload.bin for append
- SegmentWriter::append_chunk(data: &[u8]) -> Result<u32> (returns ordinal)
  - Compute CRC-32 with crc32fast
  - Write data to payload.bin
  - Record PayloadIndexRecord in vec
  - Return ordinal (0-indexed)
- SegmentWriter::finalize(self) -> Result<SegmentMeta>
  - sync_all() on payload file
  - Write payload.idx file with all index records
  - Write meta file with segment metadata (use bincode)
  - sync parent directory
  - Return SegmentMeta
- Unit tests:
  - write_single_chunk: create, append, finalize, verify files exist
  - write_multiple_chunks: append 3 chunks, verify index has 3 records
  - crc_integrity: verify written CRC matches recomputed

Update crates/memd/src/store/mod.rs to add `pub mod segment;`
